# i+i

**期間:** 2024〜（公開予定: 2025年度内）
**組織:** 個人プロジェクト（父の建築事務所）
**タグ:** #wordpress #web-development #design #frontend #migration #family-business

## 📋 レジュメモード

### プロジェクト概要

父が運営する木造住宅専門の建築事務所「i+i」のWebサイトをリニューアル。ブログ型の既存サイトを、建築事例を効果的に見せるポートフォリオサイトへ移行。WordPress環境のモダン化とコンテンツ構造の再設計を実施。

### 役割・責務

- **フルスタック開発:** デザイン・開発・コンテンツ整理を単独で担当
- **WordPress移行:** Acorn + Sage を用いてモダンな開発環境を構築
- **情報設計:** カスタムタクソノミーで建築事例の分類システムを設計・実装
- **CMS最適化:** Gutenbergブロックで編集しやすい管理画面を構築

### 技術アプローチ

- **WordPressモダンスタック:** Acorn + Sage による開発体験の近代化
- **ブロックエディタ:** Gutenbergベースのカスタムブロックでデザイン性と編集性を両立
- **カスタムタクソノミー:** 建築事例の分類と回遊性を高める独自設計
- **テーマ設計:** theme.json を活用しテーマ設定を最適化

### 課題と解決

**課題:** WordPress未経験、Gutenberg/Block/theme.json など新概念の習得、本業との並行
**解決:** 公式ドキュメント中心に段階的に学習し、本業の合間で継続開発

### 主な成果

- WordPress未経験から1年でモダンなWP開発スキルを習得
- ブログ型から訴求力の高いポートフォリオ構造への転換を実現
- デザインと編集性を両立したCMS環境を構築

### スキル

- WordPress (Acorn, Sage, Gutenberg)
- PHP, JavaScript
- UI/UX Design
- Information Architecture
- 自律的学習
- タイムマネジメント（本業と並行）

## 📖 ポートフォリオ / ストーリーモード

### i+i の背景

**恩返しのためのコード**

このプロジェクトは技術挑戦であると同時に、とても個人的な意味があった。父が営む木造住宅専門の建築事務所「i+i」。その名前には「人と人」「今と未来」をつなぐ想いがある。

### 始まりは課題の発見

父のサイトを見たとき矛盾に気づいた。美しい住宅事例があるのに、ブログ投稿に埋もれている。訪問者は事例を探し回らねばならず、実力が伝わりきっていなかった。

「このままじゃもったいない」

そう感じた瞬間、これは単なるWeb制作ではなく、父への恩返しの機会だと思った。

### 未知への挑戦

当時、WordPress経験はゼロ。Gutenberg、Block、theme.json――すべて初耳だった。それでも、父が何十年も建築を学んだように、私も学べばいい。

公式ドキュメントを読み込み、本業後の夜にWordPressと向き合う日々が始まった。

### 技術的な発見の喜び

Acorn + Sage の組み合わせに出会い、「これだ」と確信。モダンな開発体験とWordPressの柔軟性を両立できる。理解が段階的に深まる感覚が心地よかった。

Custom Taxonomy の設計では、事例をどう分類すべきか父と何度も議論。「構造」「用途」「規模」――建築家の視点とWebでの見せ方のバランスを探る時間は、技術以上に大切だった。

### デザインと機能の両立

Gutenbergブロック実装には特に注力。父が更新するとき複雑な操作を強いたくない。一方でデザインも妥協できない。

試行錯誤の末、「直感的に編集でき、デザインも崩れない」仕組みが完成。父の「これなら使えそうだね」の一言で苦労が報われた。

### 時間との戦い

本業をこなしながらの開発は想像以上に大変。週末や仕事後の数時間を積み重ね、進捗が遅いと焦りもあった。

だが、それも学び。限られた時間で何に集中し、何を効率化するか。時間管理のスキルが大きく伸びた。

### まだ見ぬゴール

今、サイトは「それっぽくなっている」段階。公開はこれからだが、形が見えてきて父の期待も感じる。

完成は単なるサイト公開ではない。育ててもらった息子が、今度は父のビジネスを支える――その答えになる。

### 学んだこと

WordPress、PHP、モダンなフロントエンド、情報設計など多くを学んだが、最大の学びは別にあった。

**制約の中でこそ、創造性は磨かれる。**

時間がない中で効率を追求し、未経験技術を習得し、父の想いとユーザーのニーズを橋渡しする。すべてがエンジニアとして、人としての成長につながった。

### これから

公開の日を楽しみにしている。父の作品がもっと多くの人に届くように。そして「ありがとう」を形にした恩返しとなるように。

コードを書くたび、ブロックを組むたび、そこには技術以上の何かがある。家族への感謝であり、次世代へのバトンでもある。

i+i――人と人をつなぐ。親子をつなぐ。過去と未来をつなぐ。

このプロジェクトは、そんなつながりの物語だ。
